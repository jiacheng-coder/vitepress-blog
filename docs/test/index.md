**单元测试**

- 定义
  - 指的是对软件中的最小测试单元进行检查和验证
  - 单元
    - 可以是一个函数
    - 可以是一个类
    - 可以是一个模块
  - 快
- 好处
  - 自动化
  - 重构 - 增加自信心
  - 减少调试时间
  - 活文档
- 三部曲
  - given - 准备数据
  - when - 触发测试动作
  - then - 验证结果
